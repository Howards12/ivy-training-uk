Program structure (10 weeks)

This course is strictly delivered as a 10-week program (within the 8–12 week standard), with weekly learning outcomes and practical labs.

  1. Week 1: AI for engineers overview, key terms, responsible AI basics, tooling setup.
  2. Week 2: Data fundamentals for AI (datasets, labeling, leakage), simple analysis lab.
  3. Week 3: ML basics (supervised/unsupervised), train/test split, evaluation, lab.
  4. Week 4: Feature engineering + model selection, baseline models, lab.
  5. Week 5: NLP & prompting fundamentals, practical prompt patterns, lab.
  6. Week 6: Working with APIs/LLMs (high level), guardrails, safety, lab.
  7. Week 7: Building AI workflows (retrieval basics, vector concepts), small RAG-style demo lab.
  8. Week 8: MLOps basics: versioning, experiments, monitoring drift, lab.
  9. Week 9: Deploying AI features (latency/cost), privacy considerations, integration lab.
  10. Week 10: Capstone: ship an AI-powered feature (demo app) + documentation + responsible AI checklist.
