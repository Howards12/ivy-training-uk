Program structure (10 weeks)

This course is strictly delivered as a 10-week program (within the 8–12 week standard), with weekly learning outcomes and practical labs.

  1. Week 1: DevOps mindset, Git workflows, branching, code review standards, repo lab setup.
  2. Week 2: CI basics: pipelines, build/test stages, artifact handling, CI lab (GitHub Actions).
  3. Week 3: CD basics: environments, approvals, release strategies, deploy lab.
  4. Week 4: Containers fundamentals (Docker), images, registries, containerize an app lab.
  5. Week 5: Orchestration overview (Kubernetes/ECS concepts), deployments/services, lab.
  6. Week 6: Infrastructure as Code (Terraform), modules/state, provisioning lab.
  7. Week 7: Observability: logs/metrics/traces, SLOs, alerts, on-call hygiene, lab.
  8. Week 8: Security in DevOps (secrets, scanning, least privilege), secure pipeline lab.
  9. Week 9: Reliability & performance: rollbacks, blue/green, canary, chaos basics, lab.
  10. Week 10: Capstone: build a CI/CD pipeline deploying a containerized app with monitoring + rollback demo.
